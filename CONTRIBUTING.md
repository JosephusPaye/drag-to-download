# Contributing

### Clone the repo and install dependencies

```sh
git clone https://github.com/JosephusPaye/drag-to-download.git
cd drag-to-download
yarn install
```

### Compile and watch

```sh
yarn dev
```

### Compile and minify

```sh
yarn build
```

The generated `dist` folder can then be loaded into Chrome as an unpacked extension.


### Format files

```sh
yarn format
```
